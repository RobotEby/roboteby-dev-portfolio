# roboteby-dev-portfolio

## Overview

The portfolio is developed using modern technologies and development practices.

## Repository Structure

```
.
├── public/          # Static resources (images, etc.)
├── src/             # Main source code
│   ├── components/  # Reusable React components
│   ├── pages/       # Main pages
│   └── css/      # CSS styles
└── package.json     # Dependencies and scripts
```

## Technologies Used

- **React**: JavaScript library for building interfaces
- **TailwindCSS**: CSS framework for rapid styling
- **Framer Motion**: Library for fluid animations
- **React DOM**: Rendering React in the DOM
- **TanStack**: Library for advanced state management

## Architecture

The project follows a modular architecture with:

1. Reusable components in `/components`
2. Organization by pages in `/pages`
3. Global and modular styles in `/css`

## Installation and Execution

1. Clone the repository:

```
git clone https://github.com/roboteby-dev-repositorio.git
cd roboteby-dev-portfolio
```

2. Install dependencies:

```
npm install
```

3. Start the development server:

```
   npm run dev
```
